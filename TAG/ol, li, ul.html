html에서 목록를 만들어주고싶을때 li 태그를 사용하게 된다.
그리고 목록을 구분시켜주는 태그가 바로 ul 태그이다.
  • 1. Python
  • 2. HTML
  • 3. SQL
  • 4. git
  • 5. github
  • 6. vs code

위에 목록들을 보면 목차별로 순서가 있는데
만약 목차가 10개가 넘어가고 목차 하나를 지우면
순서를 처음부터 다 고쳐줘야 하는 그런 상황이 생길 수 있다.
이런 상황에서 유용하게 쓸 수 있는 태그가 바로 ol 태그이다.
